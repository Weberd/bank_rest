## Установка и запуск
### 1. Клонирование репозитория
```bash
git clone https://github.com/Weberd/bank-cards.git
cd bank-cards
```

### 2. Запуск

```bash
docker-compose up -d
```

### Проверка работы
Приложение будет доступно по адресу: http://localhost:8080

Swagger UI: http://localhost:8080/swagger-ui.html
